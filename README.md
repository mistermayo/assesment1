Word Counter
============

Product Description:
--------------------

A simple web application for counting the frequency of an entered word in an entered array

Setup Instructions:
===================

1. Download folder
2. Go into terminal and type "gem install bundler"
3. Now type "bundle"
4. Type "Ruby app.rb"
5. Go to localhost:4567
